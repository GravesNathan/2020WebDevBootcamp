Key Tips or Notes

  • Can use F1 to pull up command line in virtual code studio and search for commands.
  • Looks like by using ctrl+k+s when all documents are already saved will pull up keyboard shortcut menu. I like F1 more since I can search for about any command in VSC; including keyboard shortcut menu.
  • Shortcut for Boilerplate is ! tab when in html document
  • Ctrl + Shift + p will open up command area of vs code. From there search for desired function. For example "format document"
  • I just enabled format on save option in settings. Can be broken down to per file type if needed or turn off in settings.
  • Note that only li elements, and 2 others not covered yet, are valid ul children
  • can use "lorem" tab to have filler text generated. Can use "lorem*4 or other number to get more lines of the text
  • Ctrl and / to toggle whether or not something is commented in vscode. Can also highlight a section and use to comment whole section
  • Should never have an h4 without an h3 and h2 and h1 above it
