Block vs Inline

  • Inline elements fit in alongside other elements. Block level elements take up a whole block of space.

Span and Div tags

  • div is a generic container to hold things or group things together. It is a block level element.

    span is a generic container to hold or group things together, but is inline as opposed to block level.

HTML Entities
  • HTML Entities all start with a & and end with a ; . They are used to represent things that can be hard to type in html. HTML Entities List

    Common examples are < and >. Another is the & itself.
