Block vs Inline

  • Inline elements fit in alongside other elements. Block level elements take up a whole block of space.

Span and Div tags

  • div is a generic container to hold things or group things together. It is a block level element.

    span is a generic container to hold or group things together, but is inline as opposed to block level.

HTML Entities
  • HTML Entities all start with a & and end with a ; . They are used to represent things that can be hard to type in html. HTML Entities List

    Common examples are < and >. Another is the & itself.

Semantic Elements

  • More meaningful tags to use for blocks of html code. Two main reason for this are for tools such as web crawlers to more easily identify content. The other is for accessibiliy which is helpful for screen readers and other accessibiliy tools.

      Examples
    • <section> Represents a stand alone section of a website that doesn't have a more specific html semantic element.
    • <article> A self contained composition on a site, application, etc, which is intended to be independently distributed or resuable. e.g., syndication. Exmaples may include magazine and newspaper articles, blog entry, weather widgets, etc. Articles typically use a heading (h1-h6). address tags, datetime attributes can be helpful
    • <nav> Anything on the page that provides navigation links; internal to another area of same page or external to other sources. One sample showed was using a single nav to surround an unordered list that had multiple links. He didn't use a nav for every link.
    • <main> Surrounds the main content of the page, not including items that repeat on every page such as nav bars
    • <header> introductory content and usually includes nav content
    • <footer> The footer of the page, also usually includes nav content.
    • <aside> content not directly associted or perhaps indirectly related to the documents main content. Side bars are one common use of such.
    • <summary>
    • <details>
    • <time> represents time and can contain the datetime attribute which translates into machine-readable format.
    • <figure> For captions and can containt a figcaption tag within the figure. The figure, it's caption, and it's contents are referenced as a single unit.
    • There are other as well. Overall they are essential for SEO, redability, and screen readers
